# Product Vision

## Current State
Meridian is a B2B async collaboration tool that helps distributed and hybrid teams run structured, asynchronous check-ins. Rather than scheduling live meetings for routine status updates and alignment, teams use Meridian to record short video or text responses to a shared prompt — reviewed on each person's own schedule. The product sits between Loom (unstructured async video) and Linear (structured work tracking), occupying the space of lightweight, human-centred team rituals that don't require everyone online at the same time.

The core value proposition is reducing "alignment overhead" — the cost in meetings, Slack noise, and context-switching that teams pay just to know what everyone else is working on and how they're feeling about it.
